# spider-robot

A four-legged spider robot controlled by an Arduino and a Raspberry Pi with simple AI behavior.

## 🇺🇸/🇬🇧 English

This project is a small four-legged spider robot controlled by an Arduino and a Raspberry Pi.

The Raspberry Pi runs the decision logic (AI / behavior system), while the Arduino controls the motors and low-level hardware.

The goal of the project is to create a robot capable of:

- basic movement
- reacting to sensor input
- displaying emotions on a small OLED screen

This project was developed as a school robotics project.

## Hardware

- Arduino (motor controller)
- Raspberry Pi Zero 2 W (AI and logic)
- PCA9685 servo driver
- 12× servo motors SG90
- OLED display
- Sensors
- External power supply / power bank

## 🇮🇹 Italiano

Questo progetto è un piccolo robot ragno a quattro zampe controllato da un Arduino e un Raspberry Pi.

Il Raspberry Pi esegue la logica decisionale (AI / sistema di comportamento), mentre l'Arduino controlla i motori e l'hardware di basso livello.

L'obiettivo del progetto è creare un robot capace di:

Questo progetto è stato sviluppato come progetto scolastico di robotica.
